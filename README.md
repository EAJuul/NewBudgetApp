# NewBudgetApp

A cross-platform (iOS + Android) personal budgeting app built on the YNAB
zero-based budgeting method — *give every dollar a job*. **Local-first:** all
data lives on the device; no server, no account, no network dependency.

> **Status: planning & scaffold complete — implementation not started.**
> This repository contains the full architecture, the ordered task breakdown,
> and a compiling Flutter scaffold. The next agent begins at task **M0-T01**.

## For the agent picking this up
1. Read the documents in `docs/` in numerical order (01 → 10). They are the plan.
2. Read `docs/10-agent-workflow.md` — it defines exactly how to execute tasks
   with a small context window.
3. The task list is `docs/06-task-breakdown.md` — the **single source of truth
   for progress**. Start at the lowest-numbered unchecked task whose
   dependencies are met: **M0-T01**.
4. Detailed task cards live in `tasks/`. All of M0 and `M1-T01..M1-T04` are
   written; expand the rest from `tasks/TEMPLATE.md` as you go.

## Getting started
Prerequisite: the Flutter SDK ≥ 3.24 (Dart ≥ 3.5). An SDK exists at
`/Users/emilhansen/code/flutter` but its `version` file reports `3.7.0` — verify
with `flutter --version` and upgrade if needed (task **M0-T01**).
```
flutter pub get
dart run build_runner build --delete-conflicting-outputs
flutter analyze --fatal-infos
flutter test
flutter run
```
The `android/` and `ios/` folders are generated by task **M0-T01**; until then
the app cannot be run on a device or simulator.
