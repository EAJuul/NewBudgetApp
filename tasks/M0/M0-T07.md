# M0-T07 — Verify the build workflow

- **Milestone:** M0
- **Dependencies:** M0-T06

## Goal

## Context — read first

## Steps
1. Trigger `build.yml` via **workflow dispatch** from the GitHub Actions tab.

3. Confirm the `ios` job completes `flutter build ios --release --no-codesign`.
