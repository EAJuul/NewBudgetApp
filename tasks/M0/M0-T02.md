# M0-T02 — Configure dependencies

- **Milestone:** M0
- **Dependencies:** M0-T01
- **Estimated new code:** S

## Goal
Resolve every package dependency and confirm the code-generation pipeline runs.

## Context — read first
- `pubspec.yaml` (already in the repo)
- `docs/02-architecture.md` — Tech stack

## Steps
1. `flutter pub get`.
2. `flutter pub outdated` — if a dependency has a newer **major** version,
   update its constraint in `pubspec.yaml`, re-run `pub get`, and note any API
   change that affects the docs (for example a move to Riverpod 3.x). Report
   such changes to the orchestrator.
3. `dart run build_runner build --delete-conflicting-outputs` — with no
   generated sources yet this is a no-op, but it must complete without error.

## Acceptance criteria
- [x] `flutter pub get` succeeds with no version conflict.
- [x] `build_runner` completes successfully.
- [x] Any major-version bump is reflected in `pubspec.yaml` and reported.
- [x] M0-T02 is ticked in `docs/06-task-breakdown.md`.
