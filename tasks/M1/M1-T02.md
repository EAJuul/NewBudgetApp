## Public API to produce
```dart
/// A budget month, e.g. 2026-05. Immutable.
class MonthKey implements Comparable<MonthKey> {
  const MonthKey(this.year, this.month);   // asserts month in 1..12

  factory MonthKey.fromDate(DateTime date);
  factory MonthKey.parse(String yyyyMm);   // 'YYYY-MM'; throws FormatException

  final int year;    // e.g. 2026
  final int month;   // 1..12

  MonthKey next();
  MonthKey previous();
  MonthKey addMonths(int count);           // count may be negative
  int monthsUntil(MonthKey other);         // signed difference in months
  bool contains(DateTime date);            // the date falls within this month

  String toIso();                          // 'YYYY-MM', zero-padded month

  @override bool operator ==(Object other);
  @override int get hashCode;
  @override int compareTo(MonthKey other);
  @override String toString();             // equals toIso()
}
```

## Implementation notes
- Internally convert to an absolute ordinal `year * 12 + (month - 1)` for
  `addMonths` and `monthsUntil` — this avoids off-by-one errors.
- `parse` must reject malformed input (`'2026-13'`, `'abc'`, `'2026-1'`) with a
  `FormatException`.

## Tests to write

- `next()` rolls December into the next January; `previous()` the reverse.
- `addMonths` with positive, negative, and zero counts, crossing year bounds.

- `fromDate` and `contains` for the first day, last day, and mid-month.
