## Public API to produce
```dart
/// An amount of money in integer *milliunits* (1 unit = 1000 milliunits).
/// Immutable. Money is never represented with `double`.
class Money implements Comparable<Money> {
  const Money(this.milliunits);
  const Money.zero() : milliunits = 0;

  /// Builds from whole units + milliunits: Money.of(12, 340) == 12.340 units.
  /// `units` and `milliunits` must not have conflicting signs.
  factory Money.of(int units, [int milliunits = 0]);

  final int milliunits;

  Money operator +(Money other);
  Money operator -(Money other);
  Money operator -();              // unary negate
  Money operator *(int factor);

  bool operator <(Money other);
  bool operator <=(Money other);
  bool operator >(Money other);
  bool operator >=(Money other);

  bool get isZero;
  bool get isNegative;
  bool get isPositive;

  Money abs();

  @override bool operator ==(Object other);
  @override int get hashCode;
  @override int compareTo(Money other);
  @override String toString();     // debug form, e.g. "Money(12340)"
}
```

## Implementation notes
- All arithmetic is plain `int` arithmetic on `milliunits` — no `double`.
- `Money.of(units, milliunits)` = `Money(units * 1000 + milliunits)`. `assert`
  the two arguments do not have opposite signs.
- Display formatting is **not** in this class — that is `CurrencyFormatter`, a
  later task. `toString()` is for debugging only.

## Tests to write
File: `test/core/money/money_test.dart`
- `Money(12340)` equals `Money.of(12, 340)`.
- `Money.zero().isZero` is true; `isPositive` and `isNegative` are false.
- `Money(500) + Money(250) == Money(750)`; subtraction to a negative result.
- Unary negate; `*` by zero and by a negative factor.
- Every comparison operator, including equal operands.
- `abs()` for negative, positive, and zero.
- `==` / `hashCode` consistency; usable as a `Set` / `Map` key.
- `compareTo` sorts a list correctly.
